# Online Book Review Application

## Description
Cette application serveur permet aux utilisateurs de consulter, ajouter et gérer des critiques de livres en ligne. Elle est développée en **Node.js** et **Express.js**, avec un système d'authentification utilisant **sessions** et **JWT** pour sécuriser certaines routes.  

Les utilisateurs peuvent :
- Consulter la liste de tous les livres disponibles.
- Rechercher un livre par **ISBN**, **auteur** ou **titre**.
- Lire les critiques des livres.
- S'enregistrer et se connecter en tant qu'utilisateur.
- Ajouter, modifier ou supprimer leurs propres critiques.

Le projet illustre la création d'une API RESTful et l'utilisation de **Promises** et **Async/Await** pour gérer les requêtes asynchrones.

---

## Technologies utilisées
- Node.js
- Express.js
- JSON Web Tokens (JWT)
- Express-session
- Axios (pour les appels asynchrones dans certaines tâches)
- Postman / curl (pour tester les API)
